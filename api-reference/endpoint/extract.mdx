<RequestExample>
```bash cURL
curl -X POST https://api.runmilo.ai/v1/extract \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "files=@rent-roll.pdf" \
  -F "files=@operating-statement.xlsx"
```

```python Python
import requests

response = requests.post(
    "https://api.runmilo.ai/v1/extract",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files=[
        ("files", open("rent-roll.pdf", "rb")),
        ("files", open("operating-statement.xlsx", "rb")),
    ],
)

print(response.json())
```

```javascript Node.js
const form = new FormData();
form.append("files", fs.createReadStream("rent-roll.pdf"));
form.append("files", fs.createReadStream("operating-statement.xlsx"));

const response = await fetch("https://api.runmilo.ai/v1/extract", {
  method: "POST",
  headers: { Authorization: "Bearer YOUR_API_KEY" },
  body: form,
});

const data = await response.json();
```
